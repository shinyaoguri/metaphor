# ``metaphor``

A Swift + Metal creative coding library inspired by Processing, p5.js, and openFrameworks.

## Overview

metaphor provides a `Sketch` protocol for declarative frame-based rendering with 2D/3D drawing, GPU compute, post-processing, physics, audio, and more. macOS (Apple Silicon) only.

### Essentials
