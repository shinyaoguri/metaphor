# Examples

27 example projects demonstrating various features of metaphor. Each example is a standalone Swift package.

```bash
cd Examples/<name> && swift build && swift run
```

## 2D Basics

| Example | Description |
|---------|-------------|
| [Sketch2D](https://github.com/shinyaoguri/metaphor/tree/main/Examples/Sketch2D) | Minimal 2D drawing example |
| [ShapePrimitives](https://github.com/shinyaoguri/metaphor/tree/main/Examples/ShapePrimitives) | All 2D shapes: rect, circle, triangle, polygon, arc, bezier |
| [DrawingLines](https://github.com/shinyaoguri/metaphor/tree/main/Examples/DrawingLines) | Line drawing basics |
| [ColorDemo](https://github.com/shinyaoguri/metaphor/tree/main/Examples/ColorDemo) | RGB fill, stroke, and color constructors |
| [ColorWheel](https://github.com/shinyaoguri/metaphor/tree/main/Examples/ColorWheel) | HSB color generation with continuous rotation |
| [BlendModes](https://github.com/shinyaoguri/metaphor/tree/main/Examples/BlendModes) | All 8 blend modes side-by-side |
| [TextDemo](https://github.com/shinyaoguri/metaphor/tree/main/Examples/TextDemo) | Text rendering, fonts, and alignment |
| [ScreenCapture](https://github.com/shinyaoguri/metaphor/tree/main/Examples/ScreenCapture) | Screenshot saving with `save()` |
