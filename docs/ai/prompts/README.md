# Prompt Templates For metaphor Sketches

Use these prompts with `llms-sketch.txt`, `llms.txt`, and
`docs/ai/examples-index.md` in context. Replace bracketed fields before sending.

Recommended context bundle:

- `llms-sketch.txt`
- `llms.txt`
- one or two nearby examples from `docs/ai/examples-index.md`
- the current `App.swift` if iterating

## Templates

- `generative-2d.md`: abstract 2D motion, patterns, flow fields.
- `audio-reactive.md`: microphone / sound-driven visuals.
- `shader-effect.md`: custom post effects or material shaders.
- `vj-loop.md`: live performance sketches with parameters and inputs.
- `debug-fix.md`: compiler/runtime error repair.
- `polish-iteration.md`: improve a working sketch without rewriting it.
